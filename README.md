# Northwind API (Dapper Implementation)

## Overview
This project demonstrates SQL Scripts using Dapper.

## Features implemented:
- **Category Performance Report**: JOINS and aggregations.
- **Transactional Ordering**: multi-step inserts using Dependency Injection

## Tech Stack
- .NET 8 Web API
- Dapper (Micro-ORM)
- SQL Server (Northwind Database)

## How to Run
1. Run the scripts in the `/Sql Scripts/Northwind.sql` folder.
2. Update the connection string in `appsettings.json`.
3. Run the project in Visual Studio.
